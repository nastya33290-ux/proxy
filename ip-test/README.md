# 🌍 IP Test — мини-сайт для проверки прокси

Показывает твой текущий публичный **IP-адрес** и **страну**. Удобно проверять, работает ли прокси из расширения.

## Запуск (без библиотек)
Нужен только установленный Node.js.

```bash
cd ip-test
npm start
```

Затем открой в браузере: **http://localhost:3000**

> Порт можно сменить: `PORT=8080 npm start`

## Как тестировать прокси
1. Запусти сайт и посмотри IP/страну **без прокси**.
2. Включи прокси в расширении Proxy Switcher Pro.
3. Обнови страницу (кнопка 🔄) — IP и страна должны смениться на адрес прокси.
4. Добавь `localhost` в исключения (он там по умолчанию), чтобы сам сайт открывался напрямую.

Зависимостей нет — только встроенный модуль `http` из Node.js.
