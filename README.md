# ⚡ Proxy Switcher Pro

Расширение для **Chrome** и **Microsoft Edge** (Manifest V3) для управления множеством прокси-серверов.

- Корень репозитория — версия для **Chrome**.
- Папка [`edge/`](./edge) — версия для **Microsoft Edge** (тот же функционал).
- Папка [`ip-test/`](./ip-test) — мини-сайт для проверки IP/страны.

## Возможности
- Много прокси, типы **HTTP / HTTPS / SOCKS5 / SOCKS4**
- Быстрый выбор прокси в один клик из попапа + поиск
- 🎲 Кнопка «Случайный прокси» — мгновенно переключиться на случайный
- 🌍 Кнопка «Проверить IP» — открывает страницу с твоим текущим IP/страной
- Общий тумблер ON/OFF на весь браузер (индикатор статуса на иконке)
- Прокси с логином/паролем (авторизация автоматическая)
- **Исключения (bypass):** домены, IP, маски `*.example.com`, `192.168.0.0/16`, `<local>` — любой ввод нормализуется и не ломает прокси
- Массовое добавление списком: `socks5://user:pass@ip:port`, `host:port:user:pass` и др.
- Экспорт / импорт прокси в JSON

## Установка в Chrome
1. Скачай/клонируй репозиторий.
2. Открой `chrome://extensions`.
3. Включи **Режим разработчика**.
4. **Загрузить распакованное** → выбери корневую папку репозитория.

## Установка в Microsoft Edge
1. Открой `edge://extensions`.
2. Включи **Режим разработчика** (слева внизу).
3. **Загрузить распакованное** → выбери папку **`edge`**.

## Структура
| Файл | Назначение |
|------|-----------|
| `manifest.json` | Манифест MV3, права доступа |
| `background.js` | Применение/сброс прокси, авторизация, индикатор |
| `popup.html/css/js` | Быстрый выбор прокси и вкл/выкл |
| `options.html/css/js` | Управление списком, исключениями, импорт/экспорт |
| `icons/` | Иконки 16/48/128 |
